Buildkit can be installed on any system where you want to build/test Civi. So you might install buildkit on a local dev machine, inside a Vagrantbox, or on a dedicated test server.

Generally, it is best to install buildkit once on a server. Buildkit includes caches and commands that will allow you to build multiple sites from a single installation. After configuring buildkit's amp tool, you will be able to create additional sites without directly configuring MySQL or Apache. 





  Please pay attention to amp config and amp test.
  They may ask you to perform some setup tasks (such as editing the
  Apache configuration and restarting Apache). amp test should
  succeed before continuing.
