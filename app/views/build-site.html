  Buildkit cannot modify DNS or /etc/hosts. You must update these yourself.
  


  If you use Vagrant (or any other virtualization scheme), buildkit cannot
  modify network routes or port mappings. You must update these yourself.





  You previously indicated that you want to use Vagrant. All commands must run
  inside the Vagrantbox.
  
    Configuring Jenkins to work with Vagrant
    (or any other virtualization scheme) is outside the scope of Buildkit Explorer.
