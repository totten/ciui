drush make only works with Drupal.

Basic Build performs a download and install; the system is ready for you to hack/code/test.

Scheduled Test performs a clean installation and runs tests. It's suitable for use as a scheduled job in Jenkins.

Code Review Test performs a clean installation using a proposed patch. It's suitable for integrating with Github or Gerrit code-review.

The mix of modules and libraries in {{cfg.buildkit.type}} has been picked out by the fine folks at civicrm.org.
To pick modules and libraries for {{buildkitType()}}, we will write a bash script with commands like git clone, drush dl, or wp plugin install.
To pick modules and libraries for {{buildkitType()}}, we will write a drush make file.
Big Old Git Repository of Everything: You should copy all desired code (CMS code, CiviCRM code, modules, themes, etc) into a single git repository (eg {{cfg.bogre.url}}). Buildkit will download that repository (and nothing else).
